Key insight

For most SKUs, CPC and CPM are healthy. The core issue is thin profit per order — lowering prices is the wrong first reaction. The fastest levers are a higher price, a smarter bundle selector, and compatibility/specs placed right next to the CTA. Only then: new ad angles.

✕ Don't do this
  • → Lower prices on SKUs already getting clicks and ATCs
  • → Launch new ad campaigns before fixing page economics
  • → Treat hidden HTML duplicates as a top CRO problem
  • → Push larger quantities with discounts alone — explain the use-case
✓ Do this first
  • → Thin profit/order → test higher price and better bundle framing
  • → Weak click→ATC → add compatibility/specs block + how-it-works visual
  • → Weak checkout→purchase → audit shipping/COD clarity and mobile fields
  • → Low qty/order → 2-pack default or use-case selector, not a discount
  • → Install/safety products → remove every template/copy mismatch
